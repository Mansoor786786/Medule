# My Project

## Project info

A React + Vite + TypeScript application with shadcn/ui components.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

Visit `http://localhost:8080` to view the application.

### Build

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```

### Testing

```sh
npm run test
```

### Linting

```sh
npm run lint
```

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router
- React Query
- Vitest for testing
